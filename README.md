# 易腐品库存优化

## 项目背景
生鲜零售门店需要每天决定补货量，以平衡缺货损失和库存持有成本。本项目的目标是最小化总成本（订货成本 + 持有成本 + 缺货成本），同时满足每日需求。

## 方法
- **数据清洗与聚合**：按产品类别（如 Produce）汇总每日销量、需求量、废弃量。
- **时间序列预测**：使用 **Holt-Winters 指数平滑模型** 预测未来 7 天销量（考虑周季节性）。
- **运筹优化**：基于预测结果，构建 **线性规划模型（CPLEX）**，求解每日最优补货量。决策变量为每日补货量，约束包括库存平衡，目标是最小化总成本。

## 结果
- 预测销量：约 3000～3400 件/天。
- 优化后总成本为 43,805.80 元，缺货量为 0，期末库存接近 0。
- 对比不补货策略，避免了每天约 3000 件的缺货损失。
